DEDUP LOGIC

운동별로 all-time max weight (BW는 max reps) 가 처음 달성된 날짜를 PR 시점으로 잡음. 같은 무게 여러 번 들어도 첫 달성 1회만 카운트.
예시: Machine Chest Press 56.5 / 61.0 둘 다 PR이지만 61.0이 더 높음 → 61.0×6 만 표시.
